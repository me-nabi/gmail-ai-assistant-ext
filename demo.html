📦 Installation Guide

1 Get Gemini API Key:

Visit Google AI Studio and create your free Gemini API key.

2 Load Extension:

Open Chrome → Extensions → Enable "Developer mode" → Click "Load unpacked" → Select this folder.

3 Configure API:

Click the extension icon in Chrome toolbar → Paste your API key → Click "Save Settings".

4 Start Using:

Open Gmail → Compose/Reply → Look for the "AI Reply" button with tone dropdown!

🎯 How It Works

  1. Gmail Integration: Content script detects compose windows and injects AI button
  2. Tone Selection: Click dropdown arrow to choose from 5 different communication styles
  3. Context Analysis: Extension reads email thread for intelligent response generation
  4. AI Generation: Uses Gemini 2.0 Flash model for fast, accurate responses
  5. Smart Insertion: Generated text is automatically inserted into compose field
